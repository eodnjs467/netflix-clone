## 3. API 🚀
- 로그인
  - 회원가입 POST
  - 로그인 GET
- 프로필
  - 설정된 프로필 GET
  - 시청중인 컨텐츠 목록 GET
  - 추천 콘텐츠 preview GET
  - 알림 GET
- 상세 정보
  - 콘텐츠 상세 정보 GET
  - 특정 콘텐츠 시즌 정보 GET
  - 특정 시즌 에피소드 정보 GET 
- 검색
  -  콘텐츠 검색 GET
